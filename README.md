# HH Automation Project

## Goal

Automate interactions with hh.ru and collect job market data.

## Features (planned)

* [ ] Resume auto-refresh bot
* [ ] Job listings crawler
* [ ] Data storage (DB)
* [ ] Analytics (salary, tech stack trends)
* [ ] Queue-based processing (RabbitMQ)

## Tech Stack

* Java
* Playwright (Java)
* PostgreSQL (planned)
* RabbitMQ (planned)

## Motivation

Learning Java through a real-world project involving automation, scraping, and backend architecture.
